## Project name: Retail Product Analysis Dashboard 
***Overview:***   
This project is a Power BI dashboard designed to analyze retail product sales across various regions, cities, and product categories. The dashboard provides key insights into sales performance, retailer distribution, and sales trends over time.

***Features***
•	Key Metrics: Displays total sales, units sold, retailer count, and operating margin.
•	Regional Analysis: Shows retailer distribution across different regions.
•	Product Sales Analysis: Highlights total sales by product category.
•	City-wise Sales Trends: Visualizes sales distribution across major cities.
•	Monthly Sales Trends: Tracks unit sales over months.
•	Interactive Visuals: Includes bar charts, pie charts, and line charts for better data representation.

***Data Sources***
The data used in this dashboard includes:
•	Retailer sales transactions
•	Product category-wise sales
•	Regional and city-wise distribution data

***Tools & Technologies Used***
•	Power BI: For data visualization and dashboard creation.
•	DAX (Data Analysis Expressions): For creating calculated measures and aggregations.
•	Excel/CSV: For data storage and pre-processing.

***How to Use***
1.	Open the Power BI file (.pbix) to explore the dashboard.
2.	Use filters and slicers to interact with the data.
3.	Analyze trends and insights for retail product sales.
